### Como usar os GPX

=== "Garmin"
    Copie os arquivos `.gpx` para `/Garmin/GPX/` e sincronize com BaseCamp.

=== "My Triumph App"
    1. Baixe os arquivos GPX
    2. Importe para o My Triumph App
    3. Sincronize via Bluetooth antes da partida

!!! info "Dica Técnica"
    Sempre mantenha os mapas offline do Google Maps baixados para a região, pois o sinal de celular é inexistente em vários trechos da serra.

    [Confira o guia passo a passo para baixar mapas offline](mapas-offline.md).

## Rotas de Retorno

Você tem **duas opções** de retorno a partir de Bento Gonçalves:

### Opção 1: Retorno Rápido (Padrão)

```
Bento → Vacaria → Curitiba → Ourinhos → Uberaba → Goiânia
```

- ~1.570 km em 5 dias
- Estradas melhores, mais rápido

### Opção 2: Retorno Cênico

```
Bento → São José dos Ausentes → Vacaria → Lages → Curitiba → Goiânia
```

- ~1.700 km em 5 dias
- Paisagens incríveis, revisita a serra

**Decida durante a viagem conforme seu estado de ânimo!**
